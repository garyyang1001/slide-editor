What it does

Four ways to edit a slide.

01 · DIRECT

Type to edit text

Hover any text. Hairline outline appears. Click. Type. No mode-switching.

02 · MOVE

Drag to reposition

Toggle move mode. Drag any element. Position is stored as a transform on inline style. Double-click to reset.

03 · RESIZE

Adjust font size

Focus a text element. The floating toolbar above it lets you bump size up or down 2px at a time.

04 · REWRITE

AI rewrite a block

Mark a position. Type a prompt. Run instantly via Claude Code, or queue and batch later.

A typical edit pass

The messy middle of slide work.

The editor is built for the moment when you have a draft, the slides are roughly right, and you want to iterate on copy and layout fast — without bouncing to a separate tool.

  1. Click a paragraph. Tighten the wording yourself.
  2. Stuck on phrasing? Mark it, type "make this punchier," click 立即重寫.
  3. Heading too small? Focus it, hit Alt+↑ until it feels right.
  4. Card alignment off? Switch to move mode, nudge it 8 pixels left.
  5. Press ⌘S. Done. A backup of the file went to .backups/.
